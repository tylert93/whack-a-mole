Overview

In whack-a-mole the player must hit the mole as many times as they can in 30 seconds. Every time they manage to whack the mole; they gain a point. This is made challenging by the fact that the mole changes position every 3 quarters of a second.
The gameboard is made up of 9 circles. Each circle has an ‘id’ attribute used to number them from 1 to 9. A combination of ‘Math.floor()’ and ‘Math.random()’ to generate a random number between 1 and 9. This circle with this ‘id’ is given the class of ‘mole’ using the ‘.classList.add()’ method. This makes the circles background image to a picture of a mole, displaying where the mole is to the player. A variable called ‘hitPositoin’ is created to store the ‘id’ of the current circle with the class of mole. The ‘.setInterval()’ method is used to randomise the position of the mole every 3 quarters of a second.
An event listener is added to each circle to detect when the mole has been whacked. If the ‘id’ of the clicked circle is equal to the ‘hitPosition’ then the player has whacked the mole. The player’s score is increased by 1 and displayed to the player using the ‘.textCotent’ method. When the timer reaches 0 the game is over the mole is removed from the board.
